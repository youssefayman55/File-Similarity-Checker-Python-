# File-Similarity-Checker-Python-
A simple Python project that compares the content of two text files and calculates how similar they are using the built-in difflib library.

🚀 Project Description :

This project reads two text files and measures their similarity using Python’s SequenceMatcher algorithm.

It outputs a percentage showing how closely the two files match.

🧠 How It Works :

1- Reads content from two .txt files

2- Compares them using string similarity

3- Calculates similarity ratio

4- Displays result as a percentage


🛠️ Tech Stack:

1- Python 🐍

2- difflib (SequenceMatcher)

📦 Installation":

No external installation required (difflib is built-in with Python).

🎯 Features :

1-v📂 Compare two text files

2- 📊 Get similarity percentage

3-⚡ Fast and lightweight

4- 🧠 Uses built-in Python library
